# 대회 전용 마작 레이팅 사이트 (Mahjong Tournament Rating Site)

오직 대회(Team Match) 대국 기록을 관리하고 통계를 볼 수 있도록 가볍게 개편된 독립형 마작 레이팅 웹사이트입니다.

## 커스텀 설정 안내

포크(Fork) 또는 클론(Clone) 후 본인만의 사이트로 구축하기 위해 다음 파일들을 수정해 주세요.

### 1. 사이트 이름
`app.py` 파일의 최상단(18번째 줄 근처)에 있는 `CLUB_NAME` 변수를 본인의 동아리명이나 모임 이름으로 변경하세요.
```python
CLUB_NAME = "<동아리명>"  # 이 부분을 원하는 이름으로 변경하세요
```

### 2. 아이콘 및 이미지 교체
현재 `static` 폴더에 들어있는 아이콘 파일들은 이전 설정이 남아있을 수 있습니다. 모임 성격에 맞는 아이콘(예: 마작패 이미지, 로고 등)으로 직접 덮어씌워 주셔야 합니다.
- `static/favicon.ico`: 브라우저 탭에 표시되는 기본 아이콘
- `static/icon.png`: 모바일 홈 화면 추가 등에 쓰이는 192x192 해상도 아이콘

### 3. PWA 매니페스트 수정
스마트폰에서 '홈 화면에 추가'를 할 때 표시되는 앱 이름을 변경하려면 `static/manifest.json` 파일을 수정하세요.
- `"name"` 과 `"short_name"` 항목을 원하는 이름으로 변경하시면 됩니다.

### 4. 기본 룰(Rule) 설정
`config.json` 파일에서 점수 계산 방식, 기본 점수, 우마 등을 수정할 수 있습니다.
- `"START_SCORE"`, `"RETURN_SCORE"`, `"UMA"`, `"OKA_TO_1ST"`: 점수 체계 관련
- `"RULES_TOURNAMENT"`: 대회 규칙 텍스트
